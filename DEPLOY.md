# Despliegue de la aplicación

Siga estos pasos para desplegar el backend con Serverless Framework:

1. Instale las dependencias:
   ```bash
   npm install
   ```
2. Compile el código TypeScript:
   ```bash
   npm run build
   ```
3. Despliegue en AWS:
   ```bash
   npx serverless deploy
   ```

Necesita tener configuradas sus credenciales de AWS previamente.
