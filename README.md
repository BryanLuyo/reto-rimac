# Reto Rimac

Proyecto de ejemplo de backend Serverless para agendamiento de citas médicas. Incluye una arquitectura basada en AWS Lambda, DynamoDB, SNS, SQS y EventBridge.

## Instalación

```bash
npm install
```

## Comandos

- `npm run build` – Compila el código TypeScript.

## Despliegue

Se utiliza Serverless Framework. Para desplegar:

```bash
npx serverless deploy
```

## API

Consulte la documentación OpenAPI en `docs/openapi.yaml`.
